# Installation

First, create a virtual environment:

```bash
python -m venv .venv
```

Then, activate the virtual environment:

- On Windows:

```bash
.venv\Scripts\Activate.ps1
```

- On macOS and Linux:

```bash
source .venv/bin/activate
```

Finally, install the required packages:

```bash
pip install -r requirements.txt
```
